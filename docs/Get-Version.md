# Get-Version

## SYNOPSIS

Get the current PS Version Table

## SYNTAX

```text
Get-Version
```

## DESCRIPTION

Get the current PS Version Table

## EXAMPLES

### Example 1

```powershell
PS C:\> Get-Version
```

This command returns the current PowerShell Version Table

## PARAMETERS

## INPUTS

### None

## OUTPUTS

### System.Object

Current PowerShell Version Table

## NOTES

This function works in with PowerShell 5 & PowerShell 7
